# 🤖 AI & Machine Learning Bootcamp

A comprehensive hands-on journey through the fundamentals and advanced concepts of Artificial Intelligence, Machine Learning, Data Engineering, and MLOps. This repository contains notes, exercises, projects, assignments, and capstone implementations completed during the bootcamp.

---

## 📚 Curriculum Overview

### Module 1: Introduction to AI & Problem Framing

* Understanding AI, Machine Learning, Deep Learning, and Generative AI
* Evolution of AI systems
* AI project lifecycle
* Business understanding and problem identification
* Framing ML problems:

  * Classification
  * Regression
  * Clustering

**Outcome:** Ability to identify and define real-world AI problems.

---

### Module 2: Data Collection & Data Engineering

* Data sources:

  * CSV Files
  * APIs
  * Web Scraping
* SQL Fundamentals
* ETL Concepts
* Structured vs Unstructured Data
* Data Ingestion Techniques
* Cloud Dataset Introduction

**Tools:** Python, Pandas, APIs

**Outcome:** Ability to gather and structure data for machine learning workflows.

---

### Module 3: Data Cleaning & Preprocessing

* Handling Missing Values
* Outlier Detection & Treatment
* Data Transformation
* Normalization & Standardization
* Encoding Categorical Features
* Feature Scaling Techniques
* Data Quality Assessment

**Outcome:** Preparing high-quality datasets for machine learning models.

---

### Module 4: Exploratory Data Analysis (EDA)

* Statistical Summaries
* Data Visualization
* Correlation Analysis
* Pattern Discovery
* Hypothesis Generation

**Tools:** Matplotlib, Seaborn

**Outcome:** Extract actionable insights and understand data patterns.

---

### Module 5: Machine Learning Fundamentals

* Supervised Learning
* Unsupervised Learning
* Regression Models

  * Linear Regression
  * Ridge Regression
  * Lasso Regression
* Classification Models

  * Logistic Regression
  * K-Nearest Neighbors (KNN)
  * Decision Trees
* Clustering

  * K-Means
* Model Evaluation Metrics

  * Accuracy
  * Precision
  * Recall
  * F1-Score

**Outcome:** Build foundational machine learning models.

---

### Module 6: Advanced Machine Learning

* Ensemble Learning

  * Random Forest
  * Gradient Boosting
* XGBoost
* Bias-Variance Tradeoff
* Hyperparameter Tuning

  * Grid Search
  * Random Search
* Cross Validation

**Outcome:** Build optimized and high-performance ML models.

---

### Module 7: Model Deployment & MLOps Basics

* Model Serialization

  * Pickle
  * Joblib
* API Development

  * Flask
  * FastAPI
* Deployment Workflows
* Version Control for ML
* Experiment Tracking

**Tools:** MLflow, Git

**Outcome:** Understand how machine learning models are deployed and managed in production.

---

### Module 8: Capstone AI Project

End-to-end AI project implementation covering:

* Problem Definition
* Data Collection
* Data Preprocessing
* Exploratory Data Analysis
* Model Building
* Evaluation
* Deployment
* Documentation & Presentation

#### Example Domains

* 💰 Finance (Credit Risk Prediction)
* 🛒 Retail (Demand Forecasting)
* 🏥 Healthcare (Disease Prediction)

**Outcome:** Complete industry-level AI project.
